Submitting a New Component

To get to the User Tools section, click on your user name in the upper right corner of the page and then select User Tools from the drop down menu. Once at the User Tools sections, click on the Submissions button and then click the New Submission button. The first page of the Submission Form asks you to verify your own information. Any changes here will directly update your profile information. »
